  <section>
    <p>
      Los Hooks son una característica introducida en React 16.8 que permite usar el estado y otras características de React sin escribir una clase. Los más comunes son <code>useState</code> y <code>useEffect</code>.
    </p>

    <h2>¿Qué son los Hooks?</h2>
    <p>Los Hooks te permiten:</p>
    <ul>
      <li>Usar el estado en componentes funcionales.</li>
      <li>Ejecutar efectos secundarios como llamadas a APIs o suscripciones.</li>
      <li>Compartir lógica entre componentes sin cambiar la jerarquía.</li>
    </ul>

    <h3>Ejemplo: useState</h3>
    <pre style="background: #f5f5f5; padding: 1rem; overflow-x: auto; border-radius: 4px;">
      <code class="language-js">{`import React, { useState } from 'react'

function Contador() {
const [contador, setContador] = useState(0)

return (

<div>
	<p>Has hecho clic {contador} veces.</p>
	<button onClick={() => setContador(contador + 1)}>Incrementar</button>
</div>
) }

export default Contador`}</code>
    </pre>

    <h3>Ejemplo: useEffect</h3>
    <pre style="background: #f5f5f5; padding: 1rem; overflow-x: auto; border-radius: 4px;">
      <code class="language-js">{`import React, { useState, useEffect } from 'react'

function Datos() {
const [data, setData] = useState(null)

useEffect(() => {
fetch('https://api.example.com/data')
.then(response => response.json())
.then(json => setData(json))
.catch(error => console.error(error))
}, [])

if (!data) return <p>Cargando...</p>

return (

<div>
	<h2>Datos desde la API:</h2>
	<pre>{JSON.stringify(data, null, 2)}</pre>
</div>
) }

export default Datos`}</code>
    </pre>

    <h2>Conclusión</h2>
    <p>
      Los Hooks han simplificado enormemente la forma de trabajar con el estado y los ciclos de vida de los componentes en React, haciendo que el código sea más limpio y fácil de mantener. Si aún no los has probado, te animo a experimentar con ellos en tus próximos proyectos.
    </p>
